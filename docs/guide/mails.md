# 邮件收发

## 收件箱

主界面展示所有代收邮箱的统一收件箱，包含以下功能：

### 邮件列表
- 分页加载，默认每页 20 封
- 按时间排序（最新在前）
- 显示发件人、主题摘要、日期、已读状态、星标
- 未读邮件加粗显示

### 搜索与筛选
- 关键词搜索（发件人 / 主题 / 正文）
- 按邮箱账号筛选
- 按已读/未读筛选
- 按星标筛选

### 邮件详情
点击邮件进入详情视图：
- 完整邮件头信息（发件人、收件人、抄送、日期）
- HTML / 纯文本双格式渲染
- 附件列表与下载
- 标记已读/未读、星标切换

## 发送邮件

Magicmail 也支持通过 SMTP 发送邮件：

1. 在邮箱设置中配置 SMTP 参数（SMTP 主机、端口、用户名、密码）
2. 点击「写邮件」按钮
3. 填写收件人、主题、正文，可选添加附件
4. 点击发送

::: info SMTP 配置
发送邮件需要在对应的邮箱账号中额外配置 SMTP 信息。如果未配置 SMTP，则该账号仅能收信不能发信。
:::
